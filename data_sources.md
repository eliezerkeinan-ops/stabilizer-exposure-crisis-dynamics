# Data sources and reproducibility notes

This package contains processed reproducibility datasets sufficient to reproduce the manuscript figures and tables. These are not raw database exports. The source databases and transformations are documented below.

Access date for source documentation: 2026-06-14.

## Source databases

| Symbol / field | Meaning | Source database | File(s) in package | Notes |
|---|---|---|---|---|
| `Sigma_socx_gdp` / $\Sigma$ | Public social expenditure as percent of GDP | OECD Social Expenditure Database (SOCX), public total social expenditure | `data/stabilizer_exposure_dataset_OECDsigma_BISH.csv`, `data/country_archetypes_table.csv`, `data/Fu_country_table.csv` | Narrow empirical proxy for the theoretical stable reservoir $S$. |
| `H_household_debt_gdp` / $H$ | Household credit/debt as percent of GDP | BIS Total Credit to the Non-Financial Sector, household sector | same processed tables | Used through $[H-H_c]_+$, with $H_c=60$ in the baseline pilot and $H_c=50,60,70$ in threshold sensitivity. |
| `exports_gdp` / $X$ | Exports of goods and services as percent of GDP | World Bank World Development Indicators (WDI) | `data/stabilizer_exposure_dataset_OECDsigma_BISH.csv` | Exposure proxy in the denominator of $Z_1$. |
| `gfcf_gdp` / $I$ | Gross fixed capital formation as percent of GDP | World Bank WDI | same processed tables | Investment exposure proxy. |
| `gdp_growth` / $g_Y$ | Real GDP growth | World Bank WDI | `data/dynamic_recovery_crisis_countries.csv`, `data/Fu_country_table.csv` | Used for $R_{2009}$, immediate rebound, and post-rebound growth slope. |
| `hh_cons_growth` / $g_C$ | Real household final consumption growth | World Bank WDI | `data/dynamic_recovery_crisis_countries.csv` | Used for shock-year relative insulation and consumption depth-duration. |
| `D_C_frac_2008_2015` / $D_C$ | Consumption depth-duration below trend | Processed from WDI household-consumption time series | `data/dynamic_recovery_crisis_countries.csv`, `data/country_archetypes_table.csv`, `data/Fu_country_table.csv` | Discrete empirical analogue of $\int x_+(t)dt$. |

## Derived quantities

Baseline stabilizer-to-exposure ratio:

$$
Z_1 = \frac{\Sigma}{X+I+[H-H_c]_+}, \qquad H_c=60.
$$

Threshold-sensitivity variants use $H_c=50,60,70$ and are stored as `Z1_trueH_Hc50`, `Z1_trueH_Hc60`, and `Z1_trueH_Hc70` in `data/stabilizer_exposure_dataset_OECDsigma_BISH.csv`.

Canonical Z1 note: manuscript correlations and Z1-based figures use `Z1_trueH_Hc*` from `data/stabilizer_exposure_dataset_OECDsigma_BISH.csv` as the source of truth. The derived tables `country_archetypes_table.csv`, `Fu_country_table.csv`, and `dynamic_recovery_crisis_countries.csv` carry the same canonical columns for plotting and reproduction. Shorter `Z1_Hc*` aliases are retained only for backwards compatibility and mirror the canonical values in this package.

Shock-year relative insulation:

$$
R_{2009}=g^{real}_{C,2009}-g^{real}_{Y,2009}.
$$

Post-rebound growth-slope proxy:

$$
F_u^{post}=\overline{g}_{Y,2011-2013}.
$$

Change in post-rebound growth slope:

$$
\Delta F_u = \overline{g}_{Y,2011-2013}-\overline{g}_{Y,2004-2007}.
$$


## Construction of depth-duration variables

The raw WDI processing is documented in `code/00_build_dynamic_recovery_metrics.py`. The script fetches WDI real-level series for 2000-2015:

- GDP, constant LCU: `NY.GDP.MKTP.KN`
- Household and NPISH final consumption expenditure, constant LCU: `NE.CON.PRVT.KN`
- Population: `SP.POP.TOTL`

For each country, GDP and household consumption are divided by population. A log-linear pre-crisis trend is fitted to each real per-capita level series over 2000-2007 and extrapolated through 2015. The consumption depth-duration coordinate is then computed as the annual discrete sum of positive fractional shortfalls below trend over 2008-2015:

$$
D_C=\sum_{t=2008}^{2015}
\left[\frac{C^{\mathrm{trend}}_{pc,t}-C_{pc,t}}{C^{\mathrm{trend}}_{pc,t}}\right]_+.
$$

The analogous GDP coordinate $D_Y$ replaces $C_{pc,t}$ by $Y_{pc,t}$. Recovery-time diagnostics record the first post-2007 year in which the per-capita series returns to its 2007 level and the first post-2007 year in which the raw gap to trend is non-positive.

## Sample layers

The core $Z_1$ sample contains countries with complete stabilizer/exposure inputs: $\Sigma$, $X$, $I$, $H$, $Z_1$, $R_{2009}$, $D_C$, and growth-side diagnostics. The extended WDI diagnostic sample contains countries with WDI time-series quantities needed for $D_C$ and $\Delta F_u$ but incomplete stabilizer/exposure inputs in the current processed table. Extended-sample countries are not used to support the $Z_1$ scaling claim.

The layer definitions used in the manuscript are stored in `data/data_coverage_layers.csv`.

## Toy-model data

Toy oscillator trajectories and scan summaries are generated by the Python scripts in `code/`. The model parameters are listed in the manuscript and in the scripts themselves. The toy model is parameter-transparent, not calibrated to country data.

## Reproduction workflow

Install Python dependencies listed in `requirements.txt`, then run:

```bash
# optional raw-WDI processing documentation; requires internet access
python code/00_build_dynamic_recovery_metrics.py

python code/01_make_toy_oscillator_figures.py
python code/02_make_phase_portrait_and_regime_map.py
python code/03_make_empirical_scaling_figures.py
python code/04_make_growth_forcing_figures.py
python code/hc_threshold_sensitivity.py
```

This regenerates the figure files used by the manuscript from the included processed CSVs and transparent toy-model equations.
