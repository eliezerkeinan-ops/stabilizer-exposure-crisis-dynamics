All processed data and code required to reproduce the figures and tables are included in the accompanying reproducibility package and will be deposited in a public GitHub/Zenodo repository before final publication. The package contains processed country-level datasets, figure-generation scripts, household-debt threshold-sensitivity scripts, a data dictionary, and source-access documentation. The raw source data are publicly available from the OECD Social Expenditure Database (SOCX), BIS total credit statistics, and the World Bank World Development Indicators.
